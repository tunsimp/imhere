# I'm Here - Mental Wellness App

A comprehensive web application designed to support mental health and wellness through various therapeutic tools and features.

## Features

### 1. Gratitude Journal
- Write and save daily gratitude entries
- View entries by date
- Track your gratitude journey over time

### 2. Daily Schedule & To-Do List
- Create a daily schedule with time slots
- Manage to-do lists with completion tracking
- Organize your day effectively

### 3. Mental Health Coping Strategies
- Access evidence-based coping techniques
- Filter strategies by category (breathing, grounding, physical, mental, emotional, sensory)
- Quick reference guide for when you need support

### 4. Thought Log
- Record negative thoughts and associated emotions
- Rate intensity of emotions (1-10 scale)
- Track patterns in your thinking over time

### 5. Cognitive Restructuring
- Challenge negative thoughts with evidence
- List evidence for and against your thoughts
- Develop alternative, more balanced thoughts
- Based on Cognitive Behavioral Therapy (CBT) principles

## Technology Stack

- **React 18** - Modern UI library
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Beautiful icon library
- **LocalStorage** - Client-side data persistence

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Data Storage

All data is stored locally in your browser's localStorage. This means:
- Your data stays private and never leaves your device
- Data persists between sessions
- No account or login required

## Usage Tips

- **Gratitude Journal**: Try to write at least 3 things you're grateful for each day
- **Schedule**: Plan your day the night before for better organization
- **Coping Strategies**: Bookmark strategies that work best for you
- **Thought Log**: Record thoughts as soon as you notice them for better accuracy
- **Cognitive Restructuring**: Take your time with this - it's most effective when done thoughtfully
